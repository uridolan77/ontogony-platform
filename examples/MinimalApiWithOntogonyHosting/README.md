# MinimalApiWithOntogonyHosting

Demonstrates `Ontogony.Hosting` service defaults composition in a minimal ASP.NET API.

## Run

```powershell
dotnet run --project .\examples\MinimalApiWithOntogonyHosting\MinimalApiWithOntogonyHosting.csproj
```

## Endpoints

- `GET /`
- `GET /health`
- `GET /ready`

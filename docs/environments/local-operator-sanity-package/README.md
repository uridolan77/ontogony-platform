# Ontogony First Working Environment Package

This package defines the first working local environment for the Ontogony stack.

Expected layout:

```text
C:\dev\
  ontogony-platform\
  allagma-dotnet\
  kanon-dotnet\
  conexus-dotnet\
  ontogony-frontend\
  ontogony-ui\
```

Goal:

```text
frontend operator UI
→ Allagma governed run
→ topology selection
→ Kanon topology authorization where required
→ Conexus route/model evidence
→ evaluation records
→ baseline comparison
→ frontend visibility
→ report validation
```

Boundary: this is **first working local environment**, not production readiness.

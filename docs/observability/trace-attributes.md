# Trace Attributes Catalog

This catalog documents all attributes in `OntogonySpanAttributes`.

## ontogony.trace_id

- Key: `ontogony.trace_id`
- Source: request tracing middleware
- Meaning: canonical Ontogony trace identifier used for cross-system correlation

## ontogony.operation_id

- Key: `ontogony.operation_id`
- Source: request tracing middleware
- Meaning: per-request operation identifier generated by middleware

## ontogony.actor_id

- Key: `ontogony.actor_id`
- Source: request tracing middleware from correlation headers
- Meaning: propagated actor identifier for request context

## ontogony.tenant_id

- Key: `ontogony.tenant_id`
- Source: request tracing middleware from correlation headers
- Meaning: propagated tenant identifier for request context

## ontogony.workspace_id

- Key: `ontogony.workspace_id`
- Source: request tracing middleware from correlation headers
- Meaning: propagated workspace identifier for request context

## ontogony.project_id

- Key: `ontogony.project_id`
- Source: request tracing middleware from correlation headers
- Meaning: propagated project identifier for request context

## ontogony.session_id

- Key: `ontogony.session_id`
- Source: request tracing middleware from correlation headers
- Meaning: propagated session identifier for request context

## ontogony.protocol

- Key: `ontogony.protocol`
- Source: package-level instrumentation and adapters
- Meaning: protocol name tag used by shared observability instrumentation

## ontogony.event_type

- Key: `ontogony.event_type`
- Source: package-level instrumentation and adapters
- Meaning: mechanical event type identifier for instrumentation dimensions

# No-product-semantics contract

## Forbidden semantic imports

Platform must not include product semantics from:

- Kanon ontology/canonical fact authority;
- Conexus routing/model/provider selection policy;
- Allagma workflow/human gate policy;
- Metabole SLOD/domain mapping logic;
- Aisthesis reconstructability scoring or memory graph meaning.

## Allowed neutral terms

- trace;
- correlation;
- actor;
- error;
- problem;
- idempotency;
- artifact;
- evidence reference;
- replay contract;
- outbox;
- header;
- meter;
- schema.

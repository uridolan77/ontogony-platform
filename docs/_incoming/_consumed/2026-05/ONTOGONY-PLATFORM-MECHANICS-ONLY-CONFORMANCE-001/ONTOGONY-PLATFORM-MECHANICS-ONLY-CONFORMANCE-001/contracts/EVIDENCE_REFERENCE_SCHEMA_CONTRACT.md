# Evidence reference schema contract

## Purpose

Define a neutral reference to an evidence item without interpreting producer-specific semantics.

## Neutral fields

- evidence id;
- trace id;
- producer;
- evidence type;
- occurred at;
- payload ref;
- hash/fingerprint;
- redaction status.

## Not allowed

- reconstructability grade;
- semantic decision result;
- model routing decision;
- SLOD mapping acceptance.

# SYSTEM-LEARNING-GUIDE-001

Cursor development package for consolidating Ontogony documentation into a canonical learning path.

This package is intentionally documentation-first. It does not add backend features and does not delete existing docs directly. It defines the audit, target guide architecture, implementation stages, validation checks, and review gates needed to create `docs/learn/*` without producing another contradictory documentation layer.

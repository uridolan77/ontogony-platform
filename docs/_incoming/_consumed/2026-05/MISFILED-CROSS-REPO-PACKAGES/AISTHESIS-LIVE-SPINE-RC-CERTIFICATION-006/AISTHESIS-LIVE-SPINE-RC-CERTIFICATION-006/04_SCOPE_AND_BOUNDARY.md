# Scope and boundary

## In scope

- Aisthesis evidence-spine hardening.
- Live five-service certification harness.
- Required-edge matrix v2 and fixtures.
- Aisthesis route/client coverage truth.
- Durable evaluation/Krisis job discipline.
- Producer and edge authorization hardening.
- Retention/erasure implementation design and optional first slice.
- OTel trace export first slice or explicit gate doc.
- Frontend live evidence-spine validation contract.
- Platform-level RC lock review documentation.
- Producer-repo rechecks only where Aisthesis certification reveals regressions.

## Out of scope

- New semantic policy logic inside Aisthesis.
- New model-routing logic inside Aisthesis.
- New workflow execution logic inside Aisthesis.
- New Metabole transformation semantics inside Aisthesis.
- Rebuilding producer services.
- Claiming production readiness.

## Boundary phrase to preserve

```text
Aisthesis proves what happened. It does not decide what should happen.
```

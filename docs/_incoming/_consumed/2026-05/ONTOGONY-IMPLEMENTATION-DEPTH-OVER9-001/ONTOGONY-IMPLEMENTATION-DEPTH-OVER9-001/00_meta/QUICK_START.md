# Quick start — Implementation Depth package

1. Copy this package into the target repo under `docs/_incoming/packages/`.
2. Read the root `README.md`.
3. Open the repo-specific slice in `02_repo_slices/`.
4. Use `04_cursor_prompts/GENERIC_UNPACK_PROMPT.md`.
5. Implement one slice at a time.
6. Run repo-local validation.
7. Write evidence using the template in `05_evidence_templates/`.
8. Do cross-repo closeout from `allagma-dotnet`.

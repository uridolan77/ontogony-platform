# Active intake packages

Packages listed in [`MANIFEST.md`](./MANIFEST.md) are **current or in-flight** Cursor handoffs.

## Rules

- Do not treat files here as canonical — promote durable content to `docs/contracts/`, `docs/operators/`, or `docs/packages/`.
- When a package is **done, promoted, superseded, or abandoned**, you **must** move it to [`../_consumed/<YYYY-MM>/`](../_consumed/) per [`../README.md`](../README.md#when-to-move-a-package-to-_consumed-required).
- Update [`MANIFEST.md`](./MANIFEST.md) and [`../_consumed/MANIFEST.md`](../_consumed/MANIFEST.md) on every archive.
- No zip files and no loose `.md` stubs in this folder.

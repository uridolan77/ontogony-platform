# WS03 — Consumer conformance harness

## Goal

Provide scripts product repos can run to prove mechanical contract compliance.

## Tasks

1. Create suite runner.
2. Add per-dimension checks.
3. Generate summary JSON.
4. Add fixture mode.
5. Add per-consumer task cards.

## Done when

A consumer run produces a machine-readable PASS/PARTIAL/FAIL/NOT_RUN summary.

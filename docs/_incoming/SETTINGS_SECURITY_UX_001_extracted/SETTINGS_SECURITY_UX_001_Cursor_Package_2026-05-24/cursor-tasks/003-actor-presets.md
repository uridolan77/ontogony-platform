# Cursor Task 003 — Actor Presets

## Goal

Add read-only, local-admin, system-service, and custom actor profiles.

## Steps

1. Define preset contract.
2. Add selector to Settings.
3. Apply preset explicitly.
4. Mark manual edits as custom.
5. Add tests.

## Acceptance

- Presets apply expected actor/roles.
- Current roles and recommended read-only profile are not confused.

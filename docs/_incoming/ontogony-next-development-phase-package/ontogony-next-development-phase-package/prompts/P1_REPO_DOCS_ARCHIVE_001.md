# P1 — REPO-DOCS-ARCHIVE-001

## Objective

Reduce search noise from `_incoming`, old prompt packages, and stale planning docs.

## Tasks

1. Identify active vs historical docs in each repo.
2. Add `ARCHIVED.md` to stale package roots, or move them under `docs/archive/`.
3. Update evidence/documentation indexes.
4. Do not delete historical evidence.
5. Mark old prompts as non-actionable.

## Acceptance

- Searching for active sprint/status docs surfaces current docs first.
- Stale planning packages cannot be mistaken for current instructions.

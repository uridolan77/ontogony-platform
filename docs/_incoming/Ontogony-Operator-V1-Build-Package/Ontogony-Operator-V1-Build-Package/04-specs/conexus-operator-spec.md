# Conexus Operator Spec

## Panels

### Route Preview

Route preview should show selected provider/model, fallback order, reasons, pricing/cost posture, and warnings.

### Quota Status

Quota should show current quota posture, limits, warnings, and degraded reasons.

### Model-call Evidence

Model-call detail should link to evidence links/bundle, usage/cost, route decision, and Evidence Spine.

## Safety

Do not expose raw prompts/completions unless existing Conexus policy explicitly allows it.

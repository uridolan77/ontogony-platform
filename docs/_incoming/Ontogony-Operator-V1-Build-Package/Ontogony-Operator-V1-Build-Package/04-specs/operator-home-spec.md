# Operator Home Spec

Preferred route: `/system`.

## Sections

1. Runtime summary: baseline, lock, moving-main warning, environment, evidence, quarantines.
2. Service health: Allagma, Kanon, Conexus, frontend, Evidence Spine, observability.
3. Runtime posture: persistence, model purposes, tool modes, real external blocked, downstream config presence.
4. Semantic authority: active ontology/version, decision/provenance/semantic graph status.
5. Model gateway: health, route preview availability, quota availability, evidence availability.
6. Primary actions: Start run, Runs, Gates, Evidence Spine, Kanon Decisions, Conexus Observability, Sandbox, Demo flows.

## Required degraded states

- service unavailable;
- timeout;
- CORS/browser blocked;
- missing token;
- missing ontology;
- partial registry/lock data.

# Boundaries and Safety

## Non-negotiable boundaries

- Kanon remains the only semantic authority.
- Allagma orchestrates governed execution but does not own semantic truth.
- Conexus owns model gateway behavior but does not own semantic meaning.
- Frontend operates and explains; it does not create authority.
- Evidence Spine resolves and explains; it does not mutate system truth.
- Real external tool execution remains blocked.
- Model assistance remains draft-only and non-authoritative.
- No production-readiness claim.

## Required safety language

Every evidence file and operator UI surface touched by this package must preserve:

```text
Real external tool execution remains blocked.
This is Docker-local/operator scope, not production readiness.
Model assistance is draft-only/non-authoritative where applicable.
```

authoritative

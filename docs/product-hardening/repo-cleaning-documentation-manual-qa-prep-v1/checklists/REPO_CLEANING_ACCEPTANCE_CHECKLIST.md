# Repo Cleaning Acceptance Checklist

## Code

- [ ] current main pulled
- [ ] build/typecheck passes or pre-existing failures documented
- [ ] focused tests pass
- [ ] OpenAPI/generated-client drift checked where applicable
- [ ] fixture/live/degraded semantics preserved
- [ ] no secrets
- [ ] no broad refactor

## Docs

- [ ] docs index exists or updated
- [ ] evidence docs discoverable
- [ ] release/closeout docs discoverable
- [ ] stale status drift fixed
- [ ] current docs separated from historical docs
- [ ] limitations consolidated
- [ ] standard structure followed as far as practical
- [ ] no mass reorg
